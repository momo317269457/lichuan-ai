## Context

关于我们是展示公司形象的重要页面，需要在视觉上与首页保持统一，同时提供有价值的内容和用户交互功能。

## Goals / Non-Goals

**Goals:**
- 页面风格与首页统一（深色工业风）
- 展示公司介绍和实力（资质、产能、服务能力）
- 展示完整联系方式
- 提供用户反馈/联系表单
- 响应式适配所有设备

**Non-Goals:**
- 不实现后端数据存储（表单仅前端演示）
- 不实现用户登录功能

## Decisions

### 1. 页面结构
- Hero 区：公司标题和简介
- 公司介绍：描述公司历史、规模和定位
- 实力展示：认证资质、服务能力、合作伙伴
- 联系方式：电话、邮箱、地址
- 联系表单：姓名、电话、留言提交

### 2. 数据源
- 使用 `brochure.json` 中的公司信息
- 固定展示内容（可后续扩展）

### 3. 表单功能
- 前端验证（必填项检查）
- 提交后显示成功提示
- 不涉及后端存储

## Migration Plan

1. 创建 about 页面组件
2. 实现各区块内容
3. 添加表单交互
4. 测试响应式布局
