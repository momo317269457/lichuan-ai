## Context

力川金属网站首页当前采用传统工业设计风格，视觉较为平淡。本次重新设计旨在升级为现代工业风格，在保持行业专业性的同时提升视觉冲击力和用户体验。

**当前状态：**
- 顶部导航栏使用传统布局
- Hero 区使用插画背景
- 核心优势使用三张白色圆角卡片
- 底部状态栏显示钢材行情

**技术约束：**
- 基于现有 React + TypeScript 技术栈
- 需要兼容现有组件架构
- 需要确保响应式设计

## Goals / Non-Goals

**Goals:**
- 采用现代工业风格重新设计首页
- 优化顶部导航栏布局和视觉效果
- 升级 Hero 区设计，增强视觉冲击力
- 重构核心优势卡片，提升现代感
- 优化底部状态栏，保留行情功能
- 确保响应式设计在各终端良好展示

**Non-Goals:**
- 不改变现有后端架构
- 不重新设计其他页面
- 不改变现有内容文案
- 不引入新的技术栈

## Decisions

### 1. 设计风格选择：现代工业风格
**选择：** 采用深色系工业风格为主色调
**理由：**
- 金属行业适合深色调，契合产品属性
- 深色系更能体现专业、高端的品牌形象
- 与传统浅色工业风形成差异化

### 2. 色彩方案
**选择：** 深灰/黑色主色调 + 橙色点缀
**理由：**
- 橙色与金属行业形成对比，突出品牌活力
- 与原有设计中的橙色元素保持一致性
- 符合金属/重工业的行业特征

### 3. 组件架构
**选择：** 模块化组件设计
**理由：**
- 便于后续维护和扩展
- 兼容现有 React 组件架构
- 便于响应式布局实现

### 4. 图片资源策略
**选择：** 使用 SVG 或优化后的图片资源
**理由：**
- SVG 更适合现代网页设计
- 更好的缩放性和加载性能
- 便于定制和动画效果实现

## Risks / Trade-offs

- [风险] 图片资源可能需要重新制作 → 评估现有资源是否可用，或使用占位符
- [风险] 响应式设计在某些设备上可能有问题 → 需要详细测试各断点
- [风险] 浏览器兼容性问题 → 使用兼容的 CSS 属性

## Migration Plan

1. 创建新的首页组件（保持旧版本备份）
2. 实现各模块组件
3. 测试响应式设计
4. 逐步替换上线
