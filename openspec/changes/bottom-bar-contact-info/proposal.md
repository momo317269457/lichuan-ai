# Proposal: bottom-bar-contact-info

- Change name: bottom-bar-contact-info
- Change summary: Replace the bottom bar content with contact information (phone, email, address). Include two variants: a text-based bottom bar and an icon-based bottom bar, ensuring visual consistency with the existing brand and no external font dependencies.

## Rationale
- Users benefit from quick access to contact information from any page.
- A unified bottom bar increases brand consistency and reduces cognitive load.
- SVG icons allow a scalable, offline-friendly design without external fonts.
